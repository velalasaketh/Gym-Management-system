# 🏋️ Advanced GYM Management System

An **Advanced Gym Management System** built using **HTML, CSS, and JavaScript** that helps manage gym operations such as members, trainers, membership plans, attendance, payments, and equipment.

This project provides an interactive dashboard for **Admin, Trainers, and Members** to manage gym activities efficiently.

---

## 📌 Features

### 🔐 Login System

* Role-based login:

  * Admin
  * Trainer
  * Member
* Simple authentication system for demonstration.

### 👥 Member Management

* Add and manage gym members
* Track member details
* Assign membership plans

### 🏋️ Trainer Management

* Manage trainer information
* Assign trainers to members
* View trainer details

### 📋 Membership Plans

* Multiple plans such as:

  * Basic Plan
  * Standard Plan
  * Premium Plan
  * Platinum Plan
* View plan duration, price, and features

### 💳 Payment Tracking

* Record membership payments
* Maintain payment history

### 📅 Attendance Tracking

* Track member attendance
* Monitor gym usage

### 🛠 Equipment Management

* Manage gym equipment
* Track equipment availability

### 🥤 Supplement Management

* Manage supplements and gym products

### 🔔 Notifications

* Send notifications to users

### 📊 Dashboard

* Interactive dashboard to monitor gym operations.
